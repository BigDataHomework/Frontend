# 海量数据大作业前端

克隆仓库，进入目录，修改分支
```shell
git clone git@github.com:BigDataHomework/Frontend.git
git checkout -b 自己喜欢的分支名 
```

运行

```shell
npm run serve
```

访问[https://localhost:8080/#/dashboard/analysis](https://localhost:8080/#/dashboard/analysis)，查看效果
